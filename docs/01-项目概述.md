# 项目概述

## 项目简介

**Fossify Voice Recorder** 是一个开源的Android语音录制应用，使用Kotlin语言开发。该应用提供了高质量的音频录制功能，支持多种音频格式，并具有简洁直观的用户界面。

## 主要功能

1. **音频录制**
   - 支持多种音频格式：M4A、MP3、OGG
   - 可配置的比特率和采样率
   - 实时音频可视化
   - 暂停/恢复录制功能

2. **音频播放**
   - 播放录制的音频文件
   - 播放进度控制
   - 快进/快退功能（10秒）
   - 播放列表管理

3. **文件管理**
   - 录音文件列表展示
   - 文件重命名
   - 文件删除（支持回收站）
   - 文件搜索功能

4. **其他特性**
   - 桌面小部件（Widget）
   - 深色主题支持
   - 自定义颜色主题
   - 多语言支持

## 技术栈

- **开发语言**: Kotlin
- **构建工具**: Gradle (Kotlin DSL)
- **最低SDK版本**: Android 8.0 (API 26)
- **目标SDK版本**: Android 14 (API 36)
- **编译SDK版本**: Android 14 (API 36)
- **Java版本**: Java 17

## 构建变体（Product Flavors）

项目支持三种构建变体：

1. **core**: 核心版本
2. **foss**: FOSS（自由开源软件）版本
3. **gplay**: Google Play版本

这些变体允许为不同的发布渠道构建不同的应用版本。
